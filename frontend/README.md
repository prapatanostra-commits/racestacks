**RaceStacks** is a decentralized application (dApp) built on the **Stacks Blockchain**, leveraging the security of Bitcoin through the Proof-of-Transfer (PoX) consensus mechanism. This project is specifically developed as part of the **Stacks Builder Rewards** ecosystem.

## 🚀 Overview

RaceStacks focuses on creating a seamless interaction between high-performance frontend interfaces and **Clarity Smart Contracts**. By utilizing the `@stacks/transactions` and `@stacks/connect` libraries, this repository demonstrates a full-stack implementation of Bitcoin L2 technology.

### Key Features
* **Clarity Smart Contract Integration:** Direct interaction with Stacks blockchain smart contracts.
* **Stacks Auth:** Secure authentication using Hiro Wallet or Xverse.
* **SIP-010 & SIP-009 Compatible:** Designed to handle Stacks tokens and NFTs.
* **Next.js Implementation:** Fast and SEO-friendly frontend for the Bitcoin economy.

## 🛠️ Technical Stack

- **Blockchain:** Stacks (Bitcoin Layer 2)
- **Smart Contract Language:** Clarity
- **Frontend:** Next.js 16 (App Router), React 19, TypeScript
- **Styling:** Tailwind CSS 4
- **Libraries:** - `@stacks/connect`
  - `@stacks/transactions`
  - `@stacks/network`
  - `micro-stacks`
